# Week 1: Hello World

## 作业描述

请编写一个 C++ 程序，实现以下功能：

1. 输出 "Hello, World!"
2. 接收用户输入的姓名，输出 "Hello, [姓名]!"

## 要求

- 使用标准输入输出
- 代码需要有适当的注释
- 程序能够正常编译运行

## 评分标准

| 项目 | 分值 |
|------|------|
| 程序能正确编译 | 30分 |
| 正确输出 Hello, World! | 30分 |
| 正确读取姓名并输出 | 30分 |
| 代码风格和注释 | 10分 |

**总分：100分**

## 提交要求

- 提交文件：`main.cpp`
- 截止时间：2024年3月15日 23:59
